### Scripts

To install dependencies:

```sh
yarn install
```

Start development mode:

```sh
yarn run start
```

Create production build:

```sh
yarn run build
```

Lint code:

```sh
yarn run lint
```

Format code with Prettier:

```sh
yarn run format
```
